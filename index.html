PathGL

A webGL renderer for Data Visualization

Get Started Download PathGL v0.0.0

Dynamic Shader Recompiliation

Pathgl bundles with a collection of powerful and commonly used GLSL code as well as a transpiler for building glsl expressions on the fly.

Virtual Dom

Pathgl implements a scene graph with svg semantics, so you can and should use it with prexisting libraries like D3.

Data Textures

Load in Billions of data points directly into your graphics card.
